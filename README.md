# imsa_json
Displays the live timing data from the IMSA website

## Requirements
* Python 2.7
* Python Packages (json, requests)
* Optional Python Packages (PrettyTable)

## How to run
Run python htmlimsa.py and imsa.html will be updated with the updated data
